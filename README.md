## Description

Bot for the [JavaScript Wars](https://twitter.com/JavaScriptWars) Twitter account. Daily tweets ranking the most popular JavaScript frontend frameworks.
![logo](img/logo.png)

## Setup

1. Clone the repo
2. `npm i`
3. `cp .env.example .env`
4. Fill in the `.env` file
5. `npm start`
